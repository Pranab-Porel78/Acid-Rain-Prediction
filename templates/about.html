About Project

IoT Enabled Acid Rain Prediction System

This project predicts the probability of acid rain using:

  • ESP32
  • DHT22 Sensor
  • MQ7 Sensor
  • MQ135 Sensor
  • OpenWeather Air Pollution API
  • InfluxDB
  • Flask
  • XGBoost (To be Integrated)
Objective

To monitor environmental conditions in real time and estimate acid rain probability using IoT devices and machine learning.
